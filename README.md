# PHP-Unit-MyWeeklyAllowance
Pocket money management module for teenagers (with TDD methodology)

Objective: at least 85% code coverage

## Project context: MyWeeklyAllowance
The MyWeeklyAllowance app allows parents to manage a ‘virtual wallet’ for their teenagers.
Each teenager has a pocket money account, and each parent can:
- create an account for a teenager,
- deposit money,
- record expenses,
- set an automatic weekly allowance
